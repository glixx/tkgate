Создание и удаление интерфейсов модулей

1) Выбрать вкладку "Интерфейс" главного окна для входа в режим редактора интерфейсов.

2) Double-click on module "bar" in module list to edit its interface.

3) Double-click on a port name or port property to change it, any changes will
be immediately reflected in the interface example.

4) Double-click on the blank line below the last port in the port list to add
a new port.

5) You can also add a new port by right-clicking on the edge of the interface
example and then selecting one of the "Add..." options.

6) Change the size of the interface by selecting and dragging the edge or corner
of the box.

7) Move the position of a port by selecting and dragging it.

Autogeneration of Interfaces

1) Double-clock on the module "foo" in the module list to bring it up in the inteface
editor.  It should have a blank interface with no ports.

2) To automatically generate an interface, click on the "Auto Generate" button, then
select "OK".  The generate ports on the interface will be determined from the ports
used in the module definition.

Return to Edit Mode

Click on the "Edit" tab in the main window to switch to the definition for
the inteface you have been editing.
